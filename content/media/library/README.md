# Official Media Library

Phase 10 Sprint 4 catalog of official-sourced media records.

- `catalog.json` — master asset list with provenance fields
- `developers/<slug>/` — mirrored logos/favicons + `*.asset.json`
- `projects/<slug>/` — brochures (when licensed/mirrored) + hotlink sidecars

**Policy:** Do not scrape portal screenshots. Do not mirror copyrighted gallery/floor-plan/hero binaries without permission. Hotlink registration is allowed.
