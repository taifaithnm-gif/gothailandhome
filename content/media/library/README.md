# Official Media Library

Phase 10 Sprint 4 + Phase 11 Media Factory Wave 1 catalog of official-sourced media records.

- `catalog.json` — master asset list with provenance fields
- `developers/<slug>/` — mirrored logos/favicons + `*.asset.json`
- `projects/<slug>/` — brochures, heroes, gallery/floor-plan subsets + `*.asset.json`
- `projects/<slug>/files/` — mirrored binary subsets (checksummed)

**Policy:** Official assets only. No portal screenshots or watermarked portal images. Gallery/hero/floor-plan binaries are archived with `pending_license` notes for checksum catalog; commercial reuse requires rights-holder license. See `LICENSE_REPORT.md`.
