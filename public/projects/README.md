# public/projects

## Purpose
Public static media for property projects (covers, galleries, amenity shots).

## Allowed files
- `.webp`, `.jpg`, `.png`, `.svg`
- Subfolders: `{project-slug}/`
- This `README.md`

## Naming rules
- Project folders: kebab-case slug matching `content/projects/{slug}`
- Files: `{slot}-{index}.{ext}` (zero-padded index preferred)

## Example
`the-livin-ramkhamhaeng/cover-01.webp`  
`ashton-asoke/gallery-03.webp`
