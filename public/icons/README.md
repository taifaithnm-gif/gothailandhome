# public/icons

## Purpose
UI and map icons used across filters, amenities, and navigation.

## Allowed files
- `.svg` preferred; `.png` when raster-only
- This `README.md`

## Naming rules
- Lowercase kebab-case: `{category}-{name}.svg`
- Prefer monochrome SVGs suitable for `currentColor`

## Example
`amenity-pool.svg`  
`transit-bts.svg`
