# public/developers

## Purpose
Public static media for developer brand assets (logos, covers) used on developer pages and cards.

## Allowed files
- `.svg`, `.png`, `.webp`, `.jpg`
- Optional subfolders: `{developer-slug}/`
- This `README.md`

## Naming rules
- Developer slug folders: lowercase kebab-case matching `content/developers/{slug}`
- Files: `{developer-slug}-{asset}.{ext}` or `logo.svg` / `cover.webp` inside the slug folder

## Example
`sansiri/logo.svg`  
`ananda-development-cover.webp`
