# public/cities

## Purpose
Public static assets for city landing and geography pages (heroes, maps, icons).

## Allowed files
- `.webp`, `.jpg`, `.png`, `.svg`
- Optional subfolders: `{city-slug}/`
- This `README.md`

## Naming rules
- City slug: lowercase kebab-case (`bangkok`, `phuket`)
- Files: `{city-slug}-{asset}.{ext}`

## Example
`bangkok-hero.webp`  
`phuket/map.svg`
