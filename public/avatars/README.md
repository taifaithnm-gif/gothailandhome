# public/avatars

## Purpose
Default and placeholder avatar images for agents, admins, and profile fallbacks.

## Allowed files
- `.webp`, `.png`, `.jpg`
- This `README.md`

## Naming rules
- `default-{style}.{ext}` or `agent-placeholder-{n}.{ext}`
- Prefer square (1:1) crops

## Example
`default-circle.webp`  
`agent-placeholder-01.png`
