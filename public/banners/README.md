# public/banners

## Purpose
Sitewide and campaign banner creatives (homepage, promotions, seasonal).

## Allowed files
- `.webp`, `.jpg`, `.png`, `.svg`
- This `README.md`

## Naming rules
- `{placement}-{campaign}-{yyyy-mm}.{ext}`
- Lowercase kebab-case only

## Example
`home-hero-launch-2026-07.webp`  
`sidebar-promo-membership.webp`
