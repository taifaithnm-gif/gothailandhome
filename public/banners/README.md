# public/banners

## Purpose

Sitewide and campaign banner creatives (homepage, promotions, seasonal).

## Allowed files

- Images: `.webp`, `.jpg`, `.png`, `.svg`
- This `README.md`

## Naming convention

- `{placement}-{campaign}-{yyyy-mm}.{ext}`
- Examples: `home-hero-launch-2026-07.webp`, `sidebar-promo-membership.webp`

## Owner

Marketing / Frontend team

## Validation rules

- Must comply with ADVERTISEMENT_STANDARD when used as ads
- Include desktop and mobile crops when aspect ratios differ
- No misleading claims or unverified yields in banner copy burnt into images

## Future usage

CMS-driven banner slots and A/B creative experiments.
