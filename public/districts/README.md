# public/districts

## Purpose
Public static assets for district pages (heroes, maps, local imagery).

## Allowed files
- `.webp`, `.jpg`, `.png`, `.svg`
- Optional subfolders: `{city-slug}/{district-slug}/` or `{district-slug}/`
- This `README.md`

## Naming rules
- District slug: lowercase kebab-case matching area packages
- Files: `{district-slug}-{asset}.{ext}`

## Example
`watthana-hero.webp`  
`bangkok/sathon/cover.webp`
