# public/districts

## Purpose

Public static assets for district-level pages within cities.

## Allowed files

- Images: `.webp`, `.jpg`, `.png`, `.svg`
- This `README.md`
- Subfolders: `{city-slug}/{district-slug}/` or `{district-slug}/` when city is implied

## Naming convention

- District slug: lowercase kebab-case matching area packages (`watthana`, `sathon`)
- Files: `{district-slug}-{asset}.{ext}`

## Owner

Content / SEO team

## Validation rules

- District must exist in area / district content packages
- Prefer unique local imagery over generic stock repeats
- Optimize for LCP on district landing pages

## Future usage

District guides, filters, and OG images.
