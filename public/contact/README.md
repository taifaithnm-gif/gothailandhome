# public/contact

## Purpose
Static assets for the public contact experience (maps, QR codes, channel graphics).

## Allowed files
- `.svg`, `.png`, `.webp`, `.jpg`
- Optional `.pdf` for printable contact cards
- This `README.md`

## Naming rules
- Lowercase kebab-case: `{purpose}-{variant}.{ext}`
- No spaces, underscores, or uppercase

## Example
`line-qr.png`  
`office-map.webp`
