## 1. Phase 1 measurement KPIs

| KPI | Definition |
| --- | --- |
| Consent rate | Granted / (granted + denied) for analytics consent prompts |
| Discovery sessions | Sessions with ≥1 `page_view` after consent |
| Lead conversion rate | `lead_intent_submit` / consented sessions |
| Favorite engagement rate | `favorite_toggle(add)` / consented sessions |
| Filter usage rate | `listing_filter_apply` / consented listing sessions |

## 2. Explicit non-KPIs (Phase 1)

- Revenue, commission, CRM stage velocity
- Paid media ROAS (requires separate ads go-live)
- Yield or investment performance metrics
