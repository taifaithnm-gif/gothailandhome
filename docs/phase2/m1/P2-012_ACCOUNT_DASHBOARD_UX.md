# P2-012 — Account Dashboard UX Spec

## IA

- `/{lang}/account` — hub
- `/{lang}/account/saved` — synced favorites/compare
- `/{lang}/account/searches` — saved searches
- `/{lang}/account/alerts` — notification prefs
- `/{lang}/account/sign-in` — magic link / password

## Empty states

- No saves: CTA to browse properties
- No searches: CTA to properties filters
- Signed out: sign-in form

## a11y

- `main` landmark; nav labeled; forms with labels; focus visible

## i18n keys

`account.*` in en/zh/th dictionaries.
